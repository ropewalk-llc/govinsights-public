# GovInsights Public

A static civic transparency site publishing "The School Debt Timeline: A Verbatim Record" for Chowan County, NC citizens.

## What This Is

A single-page public document compiling verbatim quotes from official Chowan County Board of Commissioner meeting minutes to show how a $50M school project (now ~$85M) was financed through USDA installment loans after voters rejected a bond referendum in 2018.

## Serving Locally

Any static file server works:

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .

# PHP
php -S localhost:8080
```

Then open http://localhost:8080
